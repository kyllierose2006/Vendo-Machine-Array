# Vendo-Machine-Array
The Smart Vendo project is a simulation of a real-world automated retail interface. At its core, the system utilizes Multidimensional Arrays to map out the machine's inventory grid, allowing for precise tracking of item placement, stock levels, and pricing logic.

Key Technical Features
Array-Driven Inventory: Uses a 2D Array (e.g., string[,] inventory) to represent the rows (A, B, C) and columns (1, 2, 3) of the machine, mirroring how a physical vending machine is organized.

Sequential Logic Processing: Implements loop structures to iterate through arrays, checking for item availability and validating user selection codes in real-time.

Transaction Algorithms: Features logic that calculates the difference between "inserted coins" and "item price," ensuring accurate change return and database updates.

C#: Demonstrates proficiency in handling Index-based data retrieval, which is a fundamental skill for more complex software engineering.
